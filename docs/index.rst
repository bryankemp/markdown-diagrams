markdown-diagrams
=================

Extract and render Mermaid diagrams from Markdown files to PNG, SVG, and PDF.

Installation
------------

.. code-block:: bash

   pip install markdown-diagrams

Quick Start
-----------

.. code-block:: bash

   markdown-diagrams extract document.md
